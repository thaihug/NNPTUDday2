Run locally:

1. Install json-server: `npm i -g json-server` (or use npx)
2. Start server: `npx json-server db.json`
3. Open `index.html` in browser (or serve with a static server)

Notes:
- Posts support soft-delete via `isDeleted: true`.
- IDs are stored as strings and auto-incremented on create.
